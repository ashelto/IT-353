MEAN is an open source Javascript Software stack. The first question I asked after reading that was what exactly is a software stack?
 After doing some research about software stacks or software solutions I found out that they are pieces of software that work in 
 conjunction with a web application. In the case of Web Development they are the programming tools, databases, and web severs among 
 others. MEAN is particularly useful because it is able to combine multiple frameworks allowing programmers to work in them all at 
 once. Thus allowing them to create highly dynamic and intuitive websites. 

A framework, specifically a web application framework, is a abstract (or knowledge base, solutions) that supports the development 
of web applications. There frameworks make development far easier through the use of additional services such as an API, security 
features, and networking features. Some examples of Web Application Frameworks are AJAX, MVC, and API. Frameworks make programming 
far easier and provide environmental features, and services that are very important to designing and developing various software 
and web applications.  Search Engine Optimization is the process of manipulating a website to increase its presence on search
engines, thus making it more popular. Websites use Search Engine Optimization to improve their chances of getting noticed when 
services related to theirs are searched (or googled). There is a fine line between what is and what is not legal when it comes 
to search engine optimization and many companies have been banned from google for exploiting their algorithms to show up higher 
on web searches.
